Pyqt5 + Arduino: Simple Car Dashboard

Serial comunication between Python and Arduino

This project has as its first objective the design of a very simple graphical interface, demonstrative, of the dashboard of a car. The second objective is to establish a proper communication between Python, Qt Designer and Arduino, through a serial communication and through a Python library called Pyqt5.

For the project, electronic materials from Arduino were used to be able to interact with the graphical interface created in Qt Designer through pyqt.

Detailed list of materials:


  • 1x Arduino UNO (cable included)
  • 1x Temperature sensor: lm35
  • 4x Pushbuttons
  • 2x 10k Potentiometers
  • 1x Protoboard
  • 6x 330 ohm resistors
  • Cables


The main objective of the project is to recreate a dashboard of a car, in which vehicle information is displayed. This dashboard will show the main information lights, such as the door open sensor, gasoline level sensor, engine temperature sensor, rpm and speed sensor, an odometer, directional lights, and stop lights.
